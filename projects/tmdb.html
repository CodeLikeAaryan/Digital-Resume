TMDB Data Analysis Dashboard

Project Overview

The TMDB Data Analysis Dashboard is designed to offer comprehensive insights into movie data from The Movie Database (TMDB). This personal project leverages advanced analytics to visualize and understand trends in movie ratings, genres, and performance metrics.

Objective

The main goal was to create an interactive dashboard that provides detailed analysis of TMDB data. This includes examining movie ratings, identifying trending genres, and visualizing box office performance across different regions.

Approach

Our approach involved:

  • Data Extraction: Gathering movie data from TMDB using their API, including ratings, genres, and box office stats.
  • Data Cleaning: Preprocessing the data to handle missing values, normalize fields, and prepare it for analysis.
  • Visualization: Designing an interactive dashboard in PowerBI to display key metrics, trends, and comparisons.
  • Insights Generation: Analyzing the visualizations to uncover trends, patterns, and insights that provide a deeper understanding of movie data.

Results

  • Developed an interactive dashboard that highlights key metrics such as average ratings, genre popularity, and regional box office performance.
  • Uncovered trends in movie ratings and genre popularity, helping to understand audience preferences.
  • Enabled users to explore movie data interactively, providing insights for movie enthusiasts and industry professionals.
