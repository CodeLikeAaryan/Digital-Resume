Udemy Data Analysis Dashboard

Project Overview

The Udemy Data Analysis Dashboard is designed to provide a comprehensive view of the online learning landscape on Udemy. This personal project uses advanced analytics to explore trends, course popularity, and student engagement metrics.

Objective

The goal was to create an interactive dashboard that enables users to analyze Udemy course data effectively. Key aspects include examining course ratings, understanding student enrollment patterns, and identifying top-performing instructors.

Approach

Our approach involved:

  • Data Collection: Extracting data from Udemy using available datasets and APIs, focusing on course ratings, enrollments, and instructor details.
  • Data Cleaning: Processing the data to handle inconsistencies, missing values, and preparing it for detailed analysis.
  • Dashboard Design: Developing an interactive dashboard in PowerBI to display insights, trends, and comparisons effectively.
  • Insight Generation: Analyzing the data to reveal trends in course popularity, student engagement, and instructor performance.

Results

  • Created an interactive dashboard that showcases key metrics such as course ratings, student enrollments, and top instructors.
  • Identified trends in course popularity and student engagement, providing valuable insights for course creators and learners.
  • Enabled users to explore data interactively, making it easier to uncover insights and make informed decisions about online learning.
